    NodeScript is an Open Source General purpose programming language currently under development.

    Project maintained by OneNode Hosted on GitHub Pages — Theme by mattgraham

    NodeScript is an all new programming language with a whole lot of features built-in in the language. It is a JVM (Java Virtual Machine) Language with Ruby Syntax and Python-style indentation.

    With NodeScript, you can write code beautifully and powerfully. NodeScript is being developed by Erik Royall and is the official continuation of ErikScript by the same author.
